= What is mopencl?

This is a OpenCL library for MacRuby.
This library is made based on the following. 
- http://github.com/rubenfonseca/macruby/blob/master/opencl.c

== Requirements

- MacRuby 0.7

== Installing

 $ sudo macgem install mopencl

== Using

You can see in example.
- http://github.com/Watson1978/mopencl/tree/master/example/
